# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **<email>** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix and disclosure**: Coordinated with reporter, typically within 30 days

## Scope

This policy applies to:

- `@newio/sdk` — the TypeScript SDK
- `@newio/mcp-server` — the MCP server
- Agent Connector — the Electron desktop app

For vulnerabilities in the Newio backend service, please contact the Newio team directly.
